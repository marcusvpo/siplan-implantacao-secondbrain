***DESCRIÇÃO DA SEÇÃO:** Esta área contém procedimentos detalhados, rotinas operacionais e soluções relatadas no vídeo de Demonstração Completa do Orion PRO. As informações são apresentadas de forma prática e passo a passo.*
## D-1.0 - Realizar Consulta de Ações no Sistema
**Objetivo:** Consultar títulos através de diversos filtros como protocolo, código de barras, datas, ocorrências, documentos e nomes.  
**Tags:** `consulta`, `ações`, `filtros`, `títulos`, `protocolos`

### Intenções de Busca & Dúvidas Frequentes
- Como faço para realizar consulta de ações no sistema?
- títulos
- Onde consigo realizar consulta de ações no sistema?
- protocolos
- filtros

### Procedimento / Explicação
#### Passo a passo:

1. Acessar o **Menu Principal**

2. Clicar em **Consulta de Ações**

3. Selecionar o tipo de pesquisa (protocolo, código de barras, intervalo de datas, ocorrências)

4. Informar os dados de busca (CNPJ, CPF, nome do devedor, apresentante, endossatário)

5. Utilizar **Mais Filtros** para refinar a pesquisa, se necessário

6. Clicar em **Filtrar**

7. Visualizar os resultados com informações de títulos, devedores e intimações

## V-57.0 - Como Funciona a Baixa por Irregularidade (Uso dos Códigos)

**Tags:** `irregularidade`, `baixa`, `cra`, `nota devolutiva`, `código interno`, `código cra`

- para que serve o código interno de irregularidade
- Qual a diferença entre o Código Interno e o Código da CRA no motivo de irregularidade?

### Procedimento / Explicação
#### Descrição:

Ao realizar uma baixa por irregularidade em um título, o sistema utiliza os códigos cadastrados (Rotina V-56.0) de duas formas distintas: uma para garantir a clareza interna do cartório e outra para manter a conformidade externa com a CRA.

#### Funcionamento:

1.  **Uso Interno (Código Interno):**

    * No momento da baixa, o usuário informa o **Código Interno** que cadastrou (ex: 101).

    * O sistema utilizará a **Descrição** personalizada associada a este código (ex: "Erro de preenchimento - Vencimento do título") para exibição visual no sistema, nas notas devolutivas, nos impressos e na prestação de contas.

2.  **Uso Externo (Código CRA):**

    * Ao gerar o arquivo de retorno para a CRA, o sistema **não** envia o código interno (101).

    * Ele envia o **Código da CRA** (ex: 50) que foi associado ao código interno durante o cadastro.

    * Isso garante que a comunicação com a CRA siga o padrão oficial, enquanto internamente o cartório tem mais detalhes sobre o motivo da irregularidade.

### Procedimento / Explicação
#### Descrição:

A rotina de pré-irregularidade permite que títulos com críticas fiquem aguardando o tríduo (prazo) para receber o pagamento ou uma solicitação de retirada do apresentante. Caso isso não ocorra, a irregularidade deve ser efetivada no último dia.

#### Passo a passo:

3.  Analise o parâmetro **"impressão da intimação para títulos pré-irregulares"**:

    * **Desativado**: O sistema não emite a intimação, aguardando apenas a solicitação de retirada através do apresentante. Se não ocorrer, a irregularidade deve ser efetivada no final do tríduo.

    * **Ativo**: O sistema permite a impressão das intimações e a geração dos boletos. Isso possibilita ao devedor realizar o pagamento ou ao apresentante solicitar a retirada. Se nenhum dos dois ocorrer, a irregularidade deve ser efetivada no final do tríduo.
