# [Hub] Gestão do OrionTN (Modelos Editor)

Ecossistema dedicado para gerenciar a criação e validação de modelos para o sistema OrionTN.

## Funcionalidades
- **Dashboard Específico:** Métricas como total de cartórios na fila, projetos bloqueados, taxa de conclusão.
- **Workspace / Editor de Modelos:** Área onde o analista faz upload de arquivos JSON, visualiza modelos e utiliza um editor de texto rico integrado para formatar peças, com salvamento automático.
